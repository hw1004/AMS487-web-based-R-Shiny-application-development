# Introduction to R
> R is a programming language for statistical computing and graphics
>
> R packages (= libraries) are collections of functions and data sets

## Installing new package form CRAN
- `install.packages("package name")`
- installing a package (e.g., ArrayTools) from the Bio conductor
  - Bioconductor uses the R statistical programming language, and is open source and open development.
  - `source(”http://Bioconductor.org/biocLite.R”)`
  - `biocLite("ArrayTools")`
- After the installation, to use the procedures in the package, you need to **import** the `library` function.
  - `library(package name)`
  - `library(ArrayTools)`
